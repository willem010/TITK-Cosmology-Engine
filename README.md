# Topologische Informatietheorie van de Kosmos (TITK) - v1.4

Welkom bij de officiële computational engine van **TITK**. Dit project benadert de evolutie en structuurvorming van de kosmos niet vanuit de traditionele deeltjesfysica of materie, maar vanuit een discreet netwerkmodel gebaseerd op data-routing en strikte informatiebudgetten.

## De Kernfilosofie: Budgetcompetitie
De dynamiek van de simulatie rust op één fundamentele netwerk-boekhoudregel:

$$W_{max} = T_{eff} + \Phi_{ext}$$

* **$W_{max}$ (Fixed Budget):** De maximale verwerkingscapaciteit per node.
* **$T_{eff}$ (Internal Processing Cost):** De interne rekenlast, die fungeert als de lokale tijddilatatieparameter. Hoe hoger de lokale informatiedichtheid, hoe hoger de vertraging.
* **$\Phi_{ext}$ (External Throughput):** De resterende capaciteit voor externe structuurvorming (in de fenomenologie zichtbaar als stervorming).

## Emergente Verklaringen voor JWST-Anomalieën
Zonder dat deze fenomenen expliciet zijn ingeprogrammeerd, lost de TITK-netwerkarchitectuur twee grote kosmologische paradoxen op die recent door de James Webb Space Telescope (JWST) zijn geobserveerd:

1. **Het Overmassieve Zwarte Gat Probleem (Little Red Dots):** In het vroege, dichte netwerk bereiken centrale nodes snel de verzadigingslimiet ($Q_{max}$, waarbij $T_{eff} = W_{max}$ en $\Phi_{ext} = 0$). Via een mechanisme van *backpressure* (akoestische terugkaatsing) wordt het omliggende netwerk lamgelegd voor externe doorvoer ($\Phi_{ext}$), waardoor zwarte gaten disproportioneel snel groeien via informatieaccretie voordat stervorming überhaupt kan beginnen.
2. **Te Oude Structuren in het Vroege Universum:** Regio's met een extreem hoge informatiedichtheid ervaren een hoge $T_{eff}$ (tijddilatatie). Vanuit een externe waarnemer (zoals JWST) lijkt het alsof deze structuren zich onmogelijk snel hebben ontwikkeld, terwijl ze in werkelijkheid simpelweg meer interne causale overgangen per universele klokcyclus hebben doorlopen.

## Status & Roadmap naar v1.5
Dit model is momenteel een functioneel prototype geschreven in JavaScript/React (v1.4) met een in-place Cooley-Tukey 2D-FFT voor de analyse van het CMB (Cosmic Microwave Background) power spectrum. 

Voor de transitie naar v1.5 werken we aan:
* Kwantitatieve mapping van $T_{eff}$ op de Schwarzschild-metriek.
* Het reproduceren van de exacte BAO-piekverhouding (1 : 2,5 : 4) door opschaling naar grotere gridresoluties lokaal (Debian).
* Formele definitie van het consolidatietrigger-criterium.
